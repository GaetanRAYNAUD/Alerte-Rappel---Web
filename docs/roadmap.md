# Idées de fonctionnalités pour Alerte Rappel

Voici une liste des fonctionnalités envisagées pour les prochaines évolutions du projet :

- **Historique de recherche :** Sauvegarde locale (LocalStorage/IndexedDB) des produits scannés pour un accès rapide.
- **OCR pour dates et numéros de lots :** Implémentation d'une analyse d'image pour extraire automatiquement les lots et dates de production sur les produits, facilitant la comparaison avec les alertes.
- **Export PDF :** Fonctionnalité permettant de générer un document PDF récapitulatif d'une alerte, en utilisant notamment les liens officiels fournis par RappelConso.
- **Notifications / Abonnements :** (Projet futur) Système permettant aux utilisateurs de s'abonner à des alertes par catégorie ou par type de produit.
